Sobre mim

Meu nome é Thiago Barros

Sou formado em Logística e trabalho como Analista de Dados em uma indústria de fertilizantes.

Atualmente, trabalho com análises de frete, mapeamento de pegada de carbono, operações de armazenagem e projetos de BI para controle de operações e resultados. Além disso atuo com projetos de Ciência de Dados, para adquirir experiência na solução de problemas de negócio e domínio sobre as ferramentas de análise de dados.

Projetos em Ciência de Dados

Desenvolvimento de um painel gerencial para negócio com Streamlit

Nesse projeto, os conceitos de programação em Pytohn, manupulação de dados, pensamento estratégico e lógica de negócio, juntos com ferramentas de desenvolvimento web como o Streamli e Github, foram utilizados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de restaurantes.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através de um link web. O painel pode ser acessado por qualquer dispositico conectado na internet.

Ferramentas utilizadas foram:

  • Python.
  • Jupyter
  • Terminal.
  • Streamlit.
  • Streamlit cloud.
  • Github.
